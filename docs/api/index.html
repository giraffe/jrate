The jRate local API documentation has not been built.

To build the API documentation locally, you must have javadoc (or a compatible tool) installed. Then, you can build the documentation by issuing the command "make api" at the top-level of your configured jRate build tree. A subsequent "make install" will install it into $DATADIR/doc/jrate/docs/api. (Your $DATADIR is most likely $PREFIX/share.)

You can also find API documentation for this release online.
