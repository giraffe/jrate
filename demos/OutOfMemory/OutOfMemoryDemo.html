jRate's OutOfMemory Demo

Description

This demo shows what is the suggested programming model to check whether or not a ScopedMemory runs out of memory while a logic is running within it. As show in this demo, the preferred way is to check for the OutOfMemoryError around the MemoryArea.enter method.

Running the demo

In order to run the demo you simply need to type the following: 

$ ./outOfMemoryDemo

The output that you'll see should be similar to the following:
